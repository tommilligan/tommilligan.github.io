Little Alchemy - Network Map

Little Alchemy is a game where you 'combine and discover new elements'. A new element is created by adding two elements together.

To visualise the network, elements and their parents were documented in XML, along with a calculated colour. From any starting elements(s), children can be calculated to the nth degree and an svg network drawn using the javascript jsNetworkX library.

This was a good project for building JQuery and ajax/XML knowledge - I aimed to host this on Github Pages, meaning no server side scripting. As one of the first truly interactive websites I have built, I learnt more about javascript's asynchronous style and control flow.

WhatColorIsX - Python Package

Simple script that takes any string, and returns a hex colour based on a Google image search.

Using the Google Custom Search API, the script submits a search query and retrieves an image, calculating the average pixel colour and returning it as a hex value. Optionally, only the hue can be returned (i.e. hsl = (x, 1.0, 0.5).

This is the first Python package I have written up neatly and posted on PyPI, which was interesting in itself. I used this module for my LAlchemy-Map project (see above) to generate colours for the network nodes.

Rain World - World Map

Rain World is an upcoming 'survival platformer set in an abandoned industrial environment ravaged by a shattered ecosystem'. In-game, the world consists of discrete rooms joined by abstract pipes.

To make a map, the exits, connections and screenshots of each room were compiled manually into an sqlite database. An force-algorithm network was generated using the python NetworkX module, and skinned using the PIL Pillow fork. The resulting large image was sliced and output in Zoomify format for web viewing with Leaflet.

This project had a few good challenges, mostly working with images bigger than single process memory allowance (~2GB). I enjoyed designing the aesthetics as well as the code, and even looked into proceedurally building the back end database by image recognition of exits (sadly ths was not to be). Hopefully I will continue to improve this project for the game's actual release (currently in alpha) to eliminate some of the manual data input, improve readability and improve efficiency of image generation.
