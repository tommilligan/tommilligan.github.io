Little Alchemy - Network Map

Little Alchemy is a game where you 'combine and discover new elements'.A new element is created by adding two elements together. Drawing a network map can easily show how to make an astronaut, or what you call a human with a tool.

 

To visualise the network, elements and their parents were documented in XML, along with a precalculated colour (using my WhatColorIsX python module).From any set of starting elements, n generations of children are added to the network and an svg representation drawn using the javascript jsNetworkX library.

This was a good project for building JQuery and ajax/XML knowledge - I aimed to host this on Github Pages, which doesn't allow server side scripting.As one of the first truly interactive websites I have built, I learnt more about javascript's asynchronous style and control flow.

WhatColorIsX - Python Module

WhatColorIsX is a simple python module that aims to answer one question well - what colour is this string? Useful for automating colour generation for multiple items, WhatColorIsX can also be used to examine local files.

 

Using the Google Custom Search API, the script submits a search query and retrieves an image, calculating the average pixel colour and returning it as a hex value.

The script can also be forced to return a bright hue (i.e. hsl = ([0.0-1.0], 1.0, 0.5)), to avoid neutral colours being returned.

This is the first Python module I have written up neatly and posted on PyPI, which allows it to be easily downloaded and used by others.I also used this as an opportunity to learn other project management skills, such as:

  • writing documentation using sphinx/ReadTheDocs.org
  • testing using nosetests/Travis-CI.org
  • checking coverage with coverage/Coveralls.io

I used this module in my LAlchemy-Map project.

Rain World - World Map

Rain World is an upcoming 'survival platformer set in an abandoned industrial environment ravaged by a shattered ecosystem'. In-game, the world consists of discrete rooms joined by abstract pipes - making it hard for a player to orientate themselves. For the severely lost, there is this map!

 

To make a map, the exits, connections and screenshots of each room were compiled manually into an sqlite database. An force-algorithm network was generated using the python NetworkX module, and skinned using the PIL Pillow fork. The resulting large image was sliced and output in Zoomify format for web viewing with Leaflet.

This was the first project I built using git and GitHub, and now understand the benefits of version control!I plan to continue using git in my future projects, and enjoy contributing on GitHub.

This project had a few good challenges, mostly working with images bigger than single process memory allowance (~2GB).Hopefully I will continue to improve this project for the game's public release; potential improvements include eliminating manual data entry, improving readability and reducing stored image size.
